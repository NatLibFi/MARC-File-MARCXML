MARC-File-MARCXML version 1.0.0
==================================

MARCXML implementation for MARC records

INSTALLATION

To install this module type the following:

    perl Makefile.PL
    make
    make test
    make install

DEPENDENCIES

This module requires these other modules and libraries:

    MARC::Record
    XML::DOM
    XML::Writer
